# 苹果的设计系统灵感

## 1.视觉主题和氛围

苹果公司的网站是控制戏剧的大师级——大片的纯黑色和近白色作为产品的电影背景，这些产品被拍摄就像画廊中的雕塑一样。设计理念归根结底是：每个像素的存在都是为了产品服务，而界面本身则退缩直至变得不可见。这并不是极简主义的审美偏好；而是极简主义。这是对物体的尊重的极简主义。

排版锚定一切。 San Francisco（用于大尺寸的 SF Pro Display，用于正文的 SF Pro Text）是 Apple 的专有字体，采用光学尺寸设计，可根据点大小自动调整字体。在显示尺寸（56 像素）、粗细 600 以及 1.07 的紧凑行高和微妙的负字母间距（-0.28 像素）下，创建的标题感觉是经过加工的，而不是排版的 — 精确、自信且毫无歉意地直接。在正文尺寸 (17px) 下，跟踪稍微放松 (-0.374px)，行高打开到 1.47，创造出舒适的阅读节奏，不会感到松弛。

色彩故事是完全二元的。产品部分在带有白色文本的纯黑色（“#000000”）背景和带有近黑色文本（“#1d1d1f”）的浅灰色（“#f5f5f7”）背景之间交替。这创造了一种电影般的节奏——黑暗部分让人感觉身临其境和优质，明亮部分感觉开放和信息丰富。唯一的色彩强调是 Apple Blue (#0071e3)，专门为交互元素保留：链接、按钮和焦点状态。这种中性色海洋中的独特强调色赋予每个可点击元素明确无误的可见性。

**主要特征：**
- 具有光学尺寸调整功能的 SF Pro 显示/文本 — 字母形式自动适应尺寸环境
- 二元亮/暗部分节奏：黑色 (`#000000`) 与浅灰色 (`#f5f5f7`) 交替
- 单一强调色：Apple Blue (`#0071e3`) 专为互动元素保留
- 纯色场上的产品英雄摄影 - 无渐变、无纹理、无干扰
- 极其严格的标题行高 (1.07-1.14)，产生压缩的、类似广告牌的效果
- 内容居中的全宽部分布局 - 视口就是画布
- 药丸形 CTA（半径 980 像素）创建柔软、平易近人的操作按钮
- 各部分之间留有充足的空白，让每个产品时刻都能呼吸

## 2. 调色板和角色

### 小学
- **纯黑** (`#000000`)：英雄部分背景，沉浸式产品展示。最暗的画布呈现最亮的产品。
- **浅灰色** (`#f5f5f7`)：备用部分背景、信息区域。不是白色——轻微的蓝灰色色调可以防止不育。
- **近黑色** (`#1d1d1f`)：浅色背景上的主要文本，深色按钮填充。比纯黑色稍暖，阅读舒适。

### 互动
- **Apple Blue** (`#0071e3`)：`--sk-focus-color`，主要 CTA 背景，对焦环。界面中唯一的彩色。
- **蓝色链接** (`#0066cc`)：`--sk-body-link-color`，内联文本链接。文本级别的可读性比 Apple Blue 稍深。
- **亮蓝色** (`#2997ff`)：深色背景上的链接。黑色部分的对比度更高的亮度。

### 文字
- **白色** (`#ffffff`)：深色背景上的文本，蓝色/深色 CTA 上的按钮文本。
- **近黑色** (`#1d1d1f`)：浅色背景上的主要正文文本。
- **黑色 80%** (`rgba(0, 0, 0, 0.8)`)：次要文本，浅色背景上的导航项目。稍微软化了。
- **黑色 48%** (`rgba(0, 0, 0, 0.48)`)：第三级文本、禁用状态、轮播控件。

### 表面和深色变体
- **深色表面 1** (`#272729`)：黑暗部分的卡片背景。
- **黑暗表面 2** (`#262628`)：黑暗环境中的微妙表面变化。
- **深色表面 3** (`#28282a`)：深色背景上的高架卡片。
- **黑暗表面 4** (`#2a2a2d`)：最高的黑暗表面海拔。
- **深色表面 5** (`#242426`)：最深的深色表面色调。

### 按钮状态
- **按钮活动** (`#ededf2`)：灯光按钮的活动/按下状态。
- **按钮默认灯** (`#fafafc`)：搜索/过滤按钮背景。
- **覆盖** (`rgba(210, 210, 215, 0.64)`)：媒体控制稀松布、覆盖。
- **白色 32%** (`rgba(255, 255, 255, 0.32)`)：深色模式关闭按钮上的悬停状态。

### 阴影
- **卡片阴影** (`rgba(0, 0, 0, 0.22) 3px 5px 30px 0px`)：产品卡片的柔和、分散的高度。偏移和宽模糊创造出自然的、摄影般的阴影。

- **显示**：`SF Pro Display`，后备：`SF Pro Icons、Helvetica Neue、Helvetica、Arial、sans-serif`

- SF Pro Display用于20px及以上； SF Pro Text 针对 19px 及以下进行了优化。

### 原则
- **光学尺寸调整作为哲学**：SF Pro 自动在显示和文本光学尺寸之间切换。显示版本具有更宽的字母间距和更细的笔划，针对大尺寸进行了优化；对于小尺寸来说，文本版本更紧凑、更坚固。这意味着字体实际上会根据上下文改变其 DNA。
- **粗细限制**：比例范围为 300（浅色）到 700（粗体），但大多数文本的宽度为 400（常规）和 600（半粗体）。粗细 300 仅出现在大型装饰文本上。粗细 700 很少见，仅用于粗体卡片标题。
- **所有尺寸的负跟踪**：与大多数仅跟踪标题的系统不同，Apple 即使在正文尺寸上也应用了微妙的负字母间距（17 像素时为 -0.374 像素，14 像素时为 -0.224 像素，12 像素时为 -0.12 像素）。这会创建普遍紧凑、高效的文本。
- **极端行高范围**：标题压缩至 1.07，而正文文本打开至 1.47，某些按钮上下文拉伸至 2.41。这种戏剧性的范围仅通过节奏就创造了清晰的视觉层次。

## 4. 组件样式

### 按钮

**原色蓝 (CTA)**
- 背景：`#0071e3`（苹果蓝）
- 文本：`#ffffff`
- 内边距：8 像素 15 像素
- 半径：8px
- 边框：1px实心透明
- 字体：SF Pro Text，17px，粗细 400
- 悬停：背景稍微变亮
- 活动：`#ededf2`背景转换
- 焦点：`2px 实心 var(--sk-focus-color, #0071E3)` 轮廓
- 使用：主要号召性用语（“购买”、“购买 iPhone”）

**原色深色**
- 背景：`#1d1d1f`
- 文本：`#ffffff`
- 内边距：8 像素 15 像素
- 半径：8px
- 字体：SF Pro Text，17px，粗细 400
- 使用：二次 CTA，深色变体

**药丸链接（了解更多/商店）**
- 背景：透明
- 文本：“#0066cc”（浅色背景）或“#2997ff”（深色背景）
- 半径：980px（全丸）
- 边框：1px实线`#0066cc`
- 字体：SF Pro Text，14px-17px
- 悬停：下划线装饰
- 使用：“了解更多”和“购买”链接 — Apple 标志性内联 CTA

**过滤/搜索按钮**
- 背景：`#fafafc`
- 文本：`rgba(0, 0, 0, 0.8)`
- 内边距：0 像素 14 像素
- 半径：11px
- 边框：3px实心`rgba(0, 0, 0, 0.04)`
- 焦点：`2px 实心 var(--sk-focus-color, #0071E3)` 轮廓
- 用途：搜索栏、过滤器控件

**媒体控制**
- 背景：`rgba(210, 210, 215, 0.64)`
- 文本：`rgba(0, 0, 0, 0.48)`
- 半径：50%（圆形）
- 活动：比例（0.9），背景偏移
- 焦点：`2px 实心 var(--sk-focus-color, #0071e3)` 轮廓，白色背景，黑色文本
- 使用：播放/暂停、旋转木马箭头

### 卡片和容器
- 背景：`#f5f5f7`（浅色）或`#272729`-`#2a2a2d`（深色）
- 边框：无（苹果系统中边框很少见）
- 半径：5px-8px
- 阴影：“rgba(0, 0, 0, 0.22) 3px 5px 30px 0px”，用于提升产品卡
- 内容：居中，填充丰富
- 悬停：没有标准悬停状态 - 卡片是静态的，其中的链接是交互式的

### 导航
- 背景：“rgba(0, 0, 0, 0.8)”（半透明深色），带有“背景滤镜：饱和(180%) 模糊(20px)”
- 高度：48px（紧凑）
- 文本：`#ffffff` 12 像素，粗细 400
- 活动：悬停时下划线
- 徽标：Apple 徽标 (SVG) 居中或左对齐，17x48px 视口
- 移动设备：通过全屏覆盖菜单折叠为汉堡包
- 导航漂浮在内容之上，无论部分背景如何，都保持其深色半透明玻璃

### 图像处理
- 纯色字段（黑色或白色）上的产品 - 没有背景，没有上下文，只有对象
- 跨越整个视口宽度的全出血部分图像
- 具有微妙阴影的极高分辨率产品摄影
- 生活方式图像仅限于圆角容器（12px+半径）

### 独特的组件

**产品英雄模块**
- 全视口宽度部分，纯色背景（黑色或“#f5f5f7”）
- 产品名称作为主要标题（SF Pro Display，56 像素，粗细 600）
- 下面的一行描述符较轻
- 两个并排的 CTA 药丸：“了解更多”（轮廓）和“购买”/“商店”（填充）

**产品网格瓷砖**
- 对比背景上的方形或近方形卡片
- 产品图像占据图块的 60-70%
- 产品名称+下面一行描述
- 底部的“了解更多”和“商店”链接对

**功能比较条**
- 产品变体的水平滚动
- 每个变体都是一张带有图像、名称和关键规格的垂直卡片
- 最少的镀铬——产品不言而喻

## 5.布局原则

### 间距系统
- 基本单位：8px
- 比例：2 像素、4 像素、5 像素、6 像素、7 像素、8 像素、9 像素、10 像素、11 像素、14 像素、15 像素、17 像素、20 像素、24 像素
- 显着特征：小尺寸（2-11 像素）时比例很密集，以 1 像素为增量，然后以更大的步长跳跃。这允许对版式和图标对齐进行精确的微调。

### 网格和容器
- 最大内容宽度：大约 980 像素（药丸按钮中反复出现的“980 像素半径”与此宽度相呼应）
- 英雄：带有居中内容块的全视口宽度部分
- 产品网格：居中容器内的 2-3 列布局
- 单栏展示英雄时刻——一款产品，一条消息，全神贯注
- 没有可见的网格线或排水沟——间距创造了隐含的结构

### 空白哲学
- **电影呼吸室**：每个产品部分占据整个视口高度（或接近它）。产品之间的空白并不是空的——它是电影中场景之间的停顿。
- **通过色块的垂直节奏**：Apple 没有单独使用间距来分隔部分，而是使用交替的背景颜色（黑色、“#f5f5f7”、白色）。每次颜色变化都标志着一个新的“场景”。
- **内部压缩，之间扩展**：文本块设置紧密（负字母间距，紧凑行高），而它们周围的空间很大。这就造成了密度和开放性之间的紧张关系。

### 边界半径比例
- Micro (5px)：小型容器、链接标签
- 标准 (8px)：按钮、产品卡、图像容器
- 舒适（11px）：搜索输入、过滤按钮
- 大 (12px)：功能面板、生活方式图像容器
- 完整药丸 (980px)：CTA 链接（“了解更多”、“商店”）、导航药丸
- 圆圈 (50%)：媒体控件（播放/暂停、箭头）

## 6. 深度和高度

|水平|治疗 |使用 |
|--------|------------|-----|
|公寓（0 级）|无阴影，纯色背景 |标准内容部分、文本块 |
|导航玻璃| `rgba(0,0,0,0.8)` 上的`背景滤镜：饱和(180%) 模糊(20px)` |粘性导航栏——玻璃效果|
|微妙提升（1 级）| `rgba(0, 0, 0, 0.22) 3px 5px 30px 0px` |产品卡、浮动元素|
|媒体控制|带有比例变换的`rgba(210, 210, 215, 0.64)`背景|播放/暂停按钮、轮播控件 |
|焦点（辅助功能）| `2px 实心 #0071e3` 轮廓 |键盘聚焦所有交互元素|

**影子哲学**：苹果极其谨慎地使用影子。主要阴影（“3px 5px 30px”，不透明度为 0.22）柔和、宽阔且偏移 — 模仿漫射工作室光在物理对象下方投射出自然阴影。这强化了“产品作为物理雕塑”的比喻。大多数元素根本没有阴影；高度来自背景颜色对比度（较暗背景上的深色卡片，或稍微不同的灰色上的浅色卡片）。

### 装饰深度
- 导航玻璃：半透明、模糊的导航栏是最容易识别的深度元素，在滚动内容之上营造出一种浮动 UI 的感觉
- 部分颜色过渡：深度是通过黑色和浅灰色部分之间的交替而不是阴影来暗示的
- 产品摄影阴影：产品本身在摄影中投射阴影，因此 UI 不需要添加合成阴影

## 7. 该做和不该做的事情

### 做
- 使用 20 像素以上的 SF Pro Display 和低于 20 像素的 SF Pro Text — 尊重光学尺寸边界
- 在所有文本大小（不仅仅是标题）上应用负字母间距 - Apple 普遍采用紧密跟踪
- 仅对交互元素使用 Apple Blue (`#0071e3`) — 它必须是单重音
- 在黑色和浅灰色 (`#f5f5f7`) 部分背景之间交替以实现电影节奏
- 对 CTA 链接使用 980 像素的药丸半径 — Apple 标志性的链接形状
- 将产品图像保留在纯色字段上，没有竞争的视觉元素
- 使用半透明深色玻璃（`rgba(0,0,0,0.8)` + 模糊）进行粘性导航
- 将标题行高压缩至 1.07-1.14 — Apple 标题非常紧凑

### 不要
- 不要引入额外的强调色——整个色彩预算都花在蓝色上
- 不要使用重阴影或多个阴影层 - Apple 的阴影系统要么是一个柔和的漫射阴影，要么什么都没有
- 不要在卡片或容器上使用边框 - Apple 几乎从不使用可见边框（特定按钮除外）
- 请勿在 SF Pro 中使用宽字母间距 — 它设计为适合各种尺寸
- 不要使用重量 800 或 900 — 最大重量为 700（粗体），即使这样也很少见
- 不要向背景添加纹理、图案或渐变 - 仅纯色
- 不要让导航变得不透明——玻璃模糊效果对于 Apple UI 标识至关重要
- 不要将正文文本居中对齐——Apple 正文文本是左对齐的；仅限头条中心
- 不要在矩形元素上使用大于 12 像素的圆角（980 像素仅适用于药丸）

### 触摸目标
- 主要 CTA：8 像素 15 像素填充，创建约 44 像素触摸高度
- 导航链接：48px 高度，有足够的间距
- 媒体控件：50% 半径圆形按钮，最小 44x44px
- “了解更多”药丸：宽大的衬垫，敲击舒适

### 崩溃策略
- 英雄标题：56px 显示 → 40px → 移动设备上 28px，按比例保持严格的行高
- 产品网格：3列→2列→单列堆叠
- 导航：全水平导航→紧凑的移动菜单（汉堡包）
- 产品英雄模块：在所有尺寸下保持全出血，文本按比例缩小
- 部分背景：在所有断点处保持全角色块 - 电影节奏永远不会中断
- 图像尺寸：产品按比例缩放，绝不裁剪 - 产品轮廓是神圣的

### 图像行为
- 产品摄影在所有断点处保持纵横比
- 英雄产品图片缩小但保持居中
- 全出血部分背景在每种尺寸下均保持不变
- 生活方式图像可能会在移动设备上裁剪，但保持圆角
- 延迟加载下屏产品图片

### 迭代指南
1. 每个交互元素都采用 Apple Blue (`#0071e3`) — 没有其他强调色
2. 部分背景交替：黑色代表沉浸式时刻，“#f5f5f7”代表信息性时刻
3. 版式光学尺寸：SF Pro Display 为 20px+，SF Pro Text 位于下方 — 切勿混合
4.所有尺寸的负字母间距：-0.28px at 56px, -0.374px at 17px, -0.224px at 14px, -0.12px at 12px
5.导航玻璃效果（半透明深色+模糊）是不可协商的——它定义了Apple网络体验
6. 产品始终出现在纯色字段上，而不会出现在英雄模块中的渐变、纹理或生活方式背景上
7. 阴影很少见并且总是柔和的：`3px 5px 30px 0.22 opacity`或者什么都没有
8. 药丸 CTA 使用 980 像素半径 — 这创建了 Apple 标志性的圆角矩形，看起来像胶囊形状
