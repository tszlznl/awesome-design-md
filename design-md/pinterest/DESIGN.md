# Pinterest 的设计系统灵感

## 1.视觉主题和氛围

Pinterest 的网站是一个温暖、灵感驱动的画布，将视觉发现视为一本生活方式杂志。该设计采用柔和、略带温暖的白色背景，并以 Pinterest 红色（“#e60023”）作为独特、大胆的品牌口音。与大多数科技平台的冷蓝色不同，Pinterest 的中性色调具有明显的温暖基调 - 灰色倾向于橄榄色/沙色（“#91918c”、“#62625b”、“#e5e5e0”）而不是冷钢，营造出一种舒适、类似工艺品的氛围，吸引人们浏览。

排版使用 Pin Sans——一种定制的专有字体，具有广泛的后备堆栈，包括日语字体，反映了 Pinterest 的全球影响力。在显示比例（70 像素，粗细 600）下，Pin Sans 可以创建大而吸引人的标题。在较小的尺寸下，系统很紧凑：按钮为 12 像素，标题为 12-14 像素。 CSS 变量命名系统（`--comp-*`、`--sema-*`、`--base-*`）揭示了复杂的三层设计令牌架构：组件级、语义级和基础级令牌。

Pinterest 的独特之处在于其宽大的边框半径系统（12px–40px，加上圆形的 50%）和暖色调的按钮背景。辅助按钮（“#e5e5e0”）具有明显温暖的沙色色调，而不是冷灰色。主要的红色按钮使用 16px 半径——圆形但不是药丸形。结合温暖的徽章背景（“hsla(60,20%,98%,.5)”——一种微妙的黄色暖色水洗）和以摄影为主的布局，其结果是一种手工制作和个人化的设计，而不是公司和枯燥的设计。

**主要特征：**
- 暖白色帆布搭配橄榄色/沙色中性色 — 舒适而不乏味
- Pinterest 红色 (`#e60023`) 作为独特的大胆口音 — 从不微妙，始终自信
- Pin Sans 自定义字体，具有全局后备堆栈（包括 CJK）
- 三层代币架构：`--comp-*` / `--sema-*` / `--base-*`
- 暖色辅助表面：沙灰色 (`#e5e5e0`)，暖色徽章 (`hsla(60,20%,98%,.5)`)
- 宽大的边框半径：标准为 16 像素，大型容器可达 40 像素
- 摄影优先的内容——图钉/图像是主要的视觉元素
- 深色近紫色文本 (`#211922`) — 温暖，带有一丝梅子色

## 2. 调色板和角色

### 主要品牌
- **Pinterest 红色** (`#e60023`)：主要 CTA，品牌口音 — 大胆、自信的红色
- **绿色 700** (`#103c25`)：`--base-color-green-700`，成功/自然口音
- **绿色 700 悬停** (`#0b2819`)：`--base-color-hover-green-700`，按绿色

### 文字
- **梅子黑** (`#211922`)：主要文本 — 温暖的近乎黑色，带有梅子底色
- **黑色** (`#000000`)：辅助文本，按钮文本
- **橄榄灰色** (`#62625b`)：次要描述，静音文本
- **暖银色** (`#91918c`)：`--comp-button-color-text-transparent-disabled`，禁用文本，输入边框
- **白色** (`#ffffff`)：深色/彩色表面上的文本

### 互动
- **焦点蓝色** (`#435ee5`)：`--comp-button-color-border-focus-outer-transparent`，焦点环
- **性能紫色** (`#6845ab`)：`--sema-color-hover-icon-performance-plus`，性能特性
- **推荐紫色**（`#7e238b`）：`--sema-color-hover-text-recommendation`，AI推荐
- **链接蓝色** (`#2b48d4`)：链接文本颜色
- **Facebook Blue** (`#0866ff`)：`--facebook-background-color`，社交登录
- **按下蓝色** (`#617bff`)：`--base-color-pressed-blue-200`，按下状态

### 表面和边框
- **沙灰色** (`#e5e5e0`)：辅助按钮背景 — 温暖、工艺般
- **暖光** (`#e0e0d9`)：圆形按钮背景、徽章
- **暖洗** (`hsla(60, 20%, 98%, 0.5)`): `--comp-badge-color-background-wash-light`, 微妙的温暖徽章背景
- **雾** (`#f6f6f3`)：光表面（不透明度为 50%）
- **边框禁用** (`#c8c8c1`)：`--sema-color-border-disabled`，禁用边框
- **悬停灰色** (`#bcbcb3`)：`--base-color-hover-grayscale-150`，悬停边框
- **深色表面** (`#33332e`)：深色部分背景

### 语义
- **红色错误** (`#9e0a0a`)：复选框/表单错误状态

## 3. 版式规则

### 字体系列
- **主要**：`Pin Sans`，后备：`-apple-system，system-ui，Segoe UI，Roboto，Oxygen-Sans，Apple Color Emoji，Segoe UI Emoji，Segoe UI Symbol，Ubuntu，Cantarell，Fira Sans，Droid Sans，Helvetica Neue，Helvetica，ヒラギノ角ゴ Pro W3，メイリオ、Meiryo、ＭＳ Ｐゴshikku、Arial`

### 层次结构

|角色 |字体|尺寸|重量 |行高|字母间距 |笔记|
|------|------|------|--------|-------------|----------------|--------|
|显示英雄 | PIN Sans | 70 像素（4.38 雷姆）| 600 |正常 |正常 |最大影响|
|章节标题 | PIN Sans | 28 像素（1.75 雷姆）| 700 |正常 | -1.2 像素 |负追踪 |
|身体| PIN Sans | 16 像素（1.00 雷姆）| 400 | 1.40 | 1.40正常 |标准阅读|
|标题粗体| PIN Sans | 14 像素（0.88 雷姆）| 700 |正常 |正常 |强大的元数据|
|标题| PIN Sans | 12 像素（0.75 雷姆）| 400–500 | 1.50 | 1.50正常 |小文字、标签 |
|按钮| PIN Sans | 12 像素（0.75 雷姆）| 400 |正常 |正常 |按钮标签|

### 原则
- **紧凑型比例**：范围为 12px–70px，具有戏剧性的跳跃 - 大多数功能文本为 12–16px，创建密集的、类似应用程序的信息层次结构。
- **温暖权重分布**：标题为 600–700，正文为 400–500。没有超轻的重量——这种类型总是给人一种厚重的感觉。
- **标题上的负跟踪**：28 像素标题上的 -1.2 像素可创建舒适、亲密的章节标题。
- **单一字体系列**：Pin Sans 处理一切 - 没有检测到辅助显示或等宽字体。

## 4. 组件样式

### 按钮

**原色红色**
- 背景：`#e60023`（Pinterest 红色）
- 文本：“#000000”（黑色 — 红色对比的不寻常选择）
- 内边距：6 像素 14 像素
- 半径：16px（大圆角，不是丸状）
- 边框：`2px 实心 rgba(255, 255, 255, 0)`（透明）
- 焦点：通过 CSS 变量实现语义边框 + 轮廓

**二次砂**
- 背景：`#e5e5e0`（暖沙灰色）
- 文本：`#000000`
- 内边距：6 像素 14 像素
- 半径：16px
- 焦点：同语义边界系统

**循环动作**
- 背景：`#e0e0d9`（暖光）
- 文本：`#211922`（梅子黑）
- 半径：50%（圆形）
- 使用：固定操作、导航控件

**幽灵/透明**
- 背景：透明
- 文本：`#000000`
- 无边框
- 使用：第三级操作

### 卡片和容器
- 摄影优先的针卡，半径大（12px–20px）
- 大多数卡片上没有传统的盒子阴影
- 白色或暖雾背景
- 某些图像容器上有 8px 白色粗边框

### 输入
- 电子邮件输入：白色背景，`1px 实心 #91918c` 边框，16px 半径，11px 15px 填充
- 重点：通过 CSS 变量实现语义边框 + 轮廓系统

### 导航
- 白色或暖色背景上的干净标题
- Pinterest 徽标 + 搜索栏居中
- Pin Sans 16px 用于导航链接
- Pinterest 红色强调活跃状态

### 图像处理
- 针式砖石网格（标志性的 Pinterest 布局）
- 圆角：图像上的 12px–20px
- 以摄影为主要内容——每个图钉都是一张图像
- 特色图像容器上的厚白色边框 (8px)

## 5.布局原则

### 间距系统
- 基本单位：8px
- 比例：4像素、6像素、7像素、8像素、10像素、11像素、12像素、16像素、18像素、20像素、22像素、24像素、32像素、80像素、100像素
- 大跳跃：部分间距 32px → 80px → 100px

### 网格和容器
- 用于引脚内容的砖石网格（签名布局）
- 居中的内容部分具有宽大的最大宽度
- 全角深色页脚
- 搜索栏作为主要导航元素

### 空白哲学
- **灵感密度**：砖石网格将销钉紧密包裹——内容密度就是价值主张。空白存在于各部分之间，而不是网格内。
- **上方呼吸，下方密度**：英雄/特色部分得到充足的填充；引脚网格紧凑且具有沉浸感。

### 边界半径比例
- 标准 (12px)：小卡片、链接
- 按钮 (16px)：按钮、输入、中卡
- 舒适（20px）：功能卡
- 大（28px）：大型容器
- 部分 (32px)：选项卡元素、大面板
- 英雄（40px）：英雄容器，大型功能块
- 圆形 (50%)：操作按钮、选项卡指示器

## 6. 深度和高度

|水平|治疗 |使用 |
|--------|------------|-----|
|公寓（0 级）|没有影子|默认 - 引脚依赖于内容，而不是阴影 |
|微妙（1 级）|最小阴影（来自代币）|提升的覆盖层、下拉菜单|
|焦点（辅助功能）| `--sema-color-border-focus-outer-default` 环 |焦点状态 |

**阴影哲学**：Pinterest 使用最少的阴影。砖石网格依靠内容（摄影）来创造视觉兴趣而不是立面效果。深度来自于表面颜色的温暖和容器的大方圆形。

## 7. 该做和不该做的事情

### 做
- 使用温暖的中性色调（`#e5e5e0`、`#e0e0d9`、`#91918c`）——温暖的橄榄色/沙色是其特征
- 仅将 Pinterest Red (`#e60023`) 用于主要 CTA — 它大胆而独特
- 专门使用 Pin Sans — 一种字体适用于所有情况
- 应用宽大的边框半径：按钮/输入为 16px，卡片为 20px+
- 保持砌体网格密集——内容密度就是价值
- 使用温暖的徽章背景（`hsla(60,20%,98%,.5)`）进行微妙的温暖洗涤
- 使用“#211922”（梅子黑）作为主要文本 - 它比纯黑色更温暖

### 不要
- 不要使用冷灰色中性色 - 始终使用暖色/橄榄色
- 不要使用纯黑色 (`#000000`) 作为主要文本 — 使用梅子黑 (`#211922`)
- 不要使用药丸形状的按钮 - 16px 半径是圆形的但不是药丸
- 不要添加浓重的阴影 - Pinterest 的设计是扁平化的，内容具有深度
- 不要在卡片上使用小边框半径（<12px）——大方的舍入是核心
- 不要引入额外的品牌颜色——红色+温暖的中性色是完整的调色板
- 不要使用较细的字体粗细 — Pin Sans 最小为 400

## 8. 响应行为

### 断点
|名称 |宽度|关键变化|
|------|--------|-------------|
|手机 | <576 像素 |单立柱，布局紧凑|
|移动大| 576–768 像素 | 2 列针网格 |
|平板电脑| 768–890 像素 |扩展网格|
|桌面小| 890–1312 像素 |标准砌体网格|
|桌面| 1312–1440 像素 |完整布局|
|大型桌面 | 1440–1680 像素 |扩展网格列 |
|超广角| >1680 像素 |最大网格密度|

### 崩溃策略
- 引脚网格：5+ 列 → 3 → 2 → 1
- 导航：搜索栏+图标→简化的移动导航
- 功能部分：并排→堆叠
- 英雄：70px → 按比例缩小
- 页脚：深色多列→堆叠

## 9. 代理提示指南

### 快速颜色参考
- 品牌：Pinterest 红色 (`#e60023`)
- 背景：白色（`#ffffff`）
- 文字：梅子黑 (`#211922`)
- 辅助文本：橄榄灰色 (`#62625b`)
- 按钮表面：沙灰色 (`#e5e5e0`)
- 边框：暖银色 (`#91918c`)
- 焦点：焦点蓝色（`#435ee5`）

### 组件提示示例
- “创建一个英雄：白色背景。标题为 70px Pin Sans 粗细 600，梅子黑色 (#211922)。红色 CTA 按钮（#e60023，16px 半径，6px 14px 填充）。辅助沙按钮（#e5e5e0，16px 半径）。”
- “设计一个 pin 卡：白色背景，16px 半径，无阴影。摄影填充顶部，16px Pin Sans 粗细 400 描述如下#62625b。”
- “构建一个圆形操作按钮：#e0e0d9 背景，50% 半径，#211922 图标。”
- “创建一个输入字段：白色背景，1px实心#91918c，16px半径，11px 15px填充。焦点：通过语义标记的蓝色轮廓。”
- “设计深色页脚：#33332e 背景。白色的 Pinterest 脚本徽标。#91918c 中的 12px Pin Sans 链接。”

### 迭代指南
1. 到处都是温暖的中性色——橄榄色/沙灰色，绝不是冷钢色
2. Pinterest Red 仅适用于 CTA——大胆而独特
3. 按钮/输入半径为 16 像素，卡片半径为 20 像素以上——慷慨但不复杂
4. Pin Sans 是唯一的字体——UI 为 12 像素，显示为 70 像素
5. 摄影承载设计——用户界面保持温暖和简约
6. 文字用梅子黑 (#211922) — 比纯黑色更温暖
