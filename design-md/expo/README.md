# 世博灵感设计系统

[DESIGN.md](https://github.com/VoltAgent/awesome-design-md/blob/main/design-md/expo/DESIGN.md) 摘自公共 [Expo](https://expo.dev/) 网站。这不是官方的设计系统。颜色、字体和间距可能不是 100% 准确。但这是构建类似东西的一个很好的起点。

## 文件

|文件|描述 |
|------|-------------|
| `设计.md` |完整的设计系统文档（9 部分）|
| `preview.html` |交互设计代币目录（轻）|
| `预览-dark.html` |互动设计代币目录（深色）|


使用 [DESIGN.md](https://github.com/VoltAgent/awesome-design-md/blob/main/design-md/expo/DESIGN.md) 用作 AI 代理（Claude、Cursor、Stitch）的参考，以生成看起来像世博会设计语言的 UI。

## 预览

使用 DESIGN.md 构建的示例登陆页面。它在一页中显示了实际的颜色、版式、按钮、卡片、间距和高度。
